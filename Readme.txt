# Blinkit Sales Analysis Dashboard 🛒

A Power BI dashboard analyzing Blinkit (India's Last Minute App) sales data spanning **2011–2022**.

---

## 📊 Overview

This report provides a comprehensive view of Blinkit's sales performance across outlet types, locations, item categories, and fat content classifications. It is designed to support data-driven decisions around inventory, outlet strategy, and product mix.

## 🗂️ Dashboard Pages

### Dashboard
The single-page interactive dashboard includes the following visuals:

| Visual | Type | Description |
|--------|------|-------------|
| **KPI Cards** | Card | High-level metrics (Total Sales, Avg Sales, Items, Ratings) |
| **Fat Content** | Donut Chart | Breakdown of sales by Regular vs Low-Fat items |
| **Fat by Outlet** | Clustered Bar Chart | Fat content distribution across outlet types |
| **Item Type** | Bar Chart | Sales performance by product category |
| **Outlet Establishment** | Line Chart | Trend of outlet openings from 2011 to 2022 |
| **Outlet Size** | Donut Chart | Sales split by Small / Medium / Large outlets |
| **Outlet Location** | Funnel Chart | Sales by Tier 1, Tier 2, and Tier 3 cities |
| **Outlet Type** | Pivot Table | Detailed breakdown by outlet type (Grocery Store, Supermarket, etc.) |
| **Slicers** | Filters | Interactive filters for Outlet Location, Outlet Size, and Item Type |

---

## 💡 Key Insights

- **Tier 3** cities contribute the highest sales volume
- **Regular fat** items dominate over low-fat alternatives
- **Supermarket Type 1** is the top-performing outlet type

---

## 🛠️ Requirements

- [Power BI Desktop](https://powerbi.microsoft.com/desktop/) (free) to open and interact with the `.pbix` file
- No external data source connection required — data is embedded in the model

---

## 🚀 Getting Started

1. Clone or download this repository
2. Open `pbix.pbix` in Power BI Desktop
3. Use the slicers (Outlet Location, Outlet Size, Item Type) to filter the dashboard
4. Hover over charts for detailed tooltips
